# 🎮 FISHKY

**FISHKY** — это головоломка на Unity, где нужно заполнять игровое поле цветными фишками, 
собирая комбинации из соседних фишек одного цвета, пока поле не переполнится.

---

## 🕹️ Геймплей

Игровое поле состоит из клеток, на которые игрок выставляет фишки разных цветов.

**Основные правила:**

- Каждые **5 секунд** случайным образом выпадает цвет очередной фишки — её нужно 
  разместить на свободной клетке поля.
- Можно **удалить фишку** нажатием на неё, но это снимает очки.
- **Комбинация засчитывается**, если 4 или более фишек одного цвета стоят рядом 
  (сверху, снизу, слева или справа друг от друга).
- Когда у фишки появляется **3 и более соседей** того же цвета, клетка последней добавленной
  фишки **временно блокируется** и становится недоступной для игрока.
- За каждую фишку в засчитанной комбинации начисляется **+1 очко**.
- За удаление фишки очки **снимаются**.

**Условия окончания игры:**

- 🔴 Всё поле заполнено фишками — игра окончена.
- 🔴 Счёт опускается до **−3** — игра окончена.

---

## 🎯 Стратегия

Старайся заранее планировать расположение фишек, чтобы собирать длинные цепочки 
одного цвета. Не злоупотребляй удалением: каждое удаление штрафует по очкам и 
приближает к проигрышу.

## 📦 Установка и запуск

### Требования

- [Unity Hub](https://unity.com/download)
- Unity **2022.2.21f1** (установить через Unity Hub)
- Android Build Support (для сборки под Android)

### Шаги

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/AlexTihonow/FISHKY.git
   ```
2. Откройте **Unity Hub** → **Open Project** → укажите папку с проектом.
3. Дождитесь импорта пакетов.
4. Нажмите **Play** для запуска в редакторе.

### Сборка под Android

1. Откройте **File → Build Settings**.
2. Выберите платформу **Android** → **Switch Platform**.
3. Нажмите **Build and Run**.
